please execute the following instructions in a terminal to install MangaScrap's dependencies for debian based systems

sudo apt-add-repository ppa:brightbox/ruby-ng
sudo apt-get update
sudo apt-get install ruby-dev ruby2.3 ruby2.3-dev libsqlite3-dev build-essential patch zlib1g-dev liblzma-dev
sudo gem install sqlite3
sudo gem install nokogiri
sudo gem install colorize
